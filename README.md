## 生成方法

首先需要安装 rouge：

```sh
gem install rouge
```

### 自动

```sh
python3 generate.py
```

### 手动

1. 获取 available themes：

    ```sh
    rougify help style
    ```

2. 选择一个喜欢的 theme，然后输出到 CSS 文件：

    ```sh
    rougify style monokai.sublime > monokai.sublime.css
    ```
